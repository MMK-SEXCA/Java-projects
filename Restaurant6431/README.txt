Steps to Run the Project on Linux Machine:
1. untar the submitted folder.
2. goto restaurant6431/bin folder
3. java Emulator <absolute path of the data file.>
	ex: java Emulator ~/Documents/Restaurant6431/data1.txt

Steps to run the Project on Eclipse:
1. Untar the submitted folder.
2. Import the project into Ecplise.
3. Click on Run -> Run Configuartion -> Java Application (Side Tab) -> (Select Emulator) -> Arguments (Tab)
4. Enter the Program arguments:  data2.txt
5. Click on Apply.
6. Click on Run.
